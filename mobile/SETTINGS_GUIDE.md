### 2. Security Settings
**Location:** Account → Security

#### 2.1 Two-Factor Authentication (2FA)
- **Purpose:** Add extra security layer to your account
- **How to enable:**
  1. Go to Settings → Security
  2. Toggle "Two-Factor Authentication" ON
  3. Download an authenticator app (Google Authenticator, Microsoft Authenticator, or Authy)
  4. Scan the QR code displayed
  5. Enter the 6-digit code from your authenticator
  6. Confirmation: 2FA is now enabled

#### 2.2 Change Password
- **Purpose:** Update your account password for security
- **Requirements:**
  - Minimum 8 characters
  - Must match confirmation password
  - Current password verification required
- **Process:**
  1. Go to Settings → Security
  2. Tap "Change Password"
  3. Enter current password
  4. Enter new password
  5. Confirm new password
  6. Tap "Update Password"

#### 2.3 Active Sessions
- **Purpose:** View and manage devices logged into your account
- **Information shown:**
  - Device type (Android Phone, etc.)
  - Current session status
  - IP address
  - Login time
- **Action:** Logout from any device remotely

#### 2.4 Login Alerts
- **Purpose:** Get notified whenever someone logs into your account
- **Toggle:** ON/OFF in Security Settings
- **Notification method:** Push notification on your device

#### 2.5 Trusted Devices
- **Purpose:** Devices that don't require 2FA on every login
- **Management:** Add/remove devices from trusted list
- **Security:** Still protected, but faster login experience

#### 2.6 Account Recovery
- **Purpose:** Regain access if you lose your password or device
- **Recovery options:**
  - Recovery email address
  - Recovery phone number
  - Backup security questions
  - Recovery codes

## Troubleshooting

### Issue: Dark Mode not changing
- **Solution:** 
  1. Close and reopen the app
  2. Check if permissions are granted
  3. Ensure app is up to date

### Issue: Can't change password
- **Solution:**
  1. Verify you're entering correct current password
  2. New password must be 8+ characters
  3. Ensure passwords match in confirmation field
  4. Check internet connection

### Issue: 2FA not working
- **Solution:**
  1. Check device time/date is correct
  2. Authenticator app time should match device time
  3. Try waiting a few seconds and entering new code
  4. Regenerate QR code if needed

### Issue: Not receiving notifications
- **Solution:**
  1. Check Push Notifications toggle is ON
  2. Verify app has permission to send notifications
  3. Check device notification settings
  4. Ensure app is not running in low-power mode
