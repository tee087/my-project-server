## 🆘 Troubleshooting

**Dark mode not working?**
- Close and reopen app
- Check app is updated

**Can't change password?**
- Ensure new password is 8+ chars
- Check it matches confirmation
- Verify current password is correct

**Need support?**
- Check FAQ first
- Email support team
- Average response: 24-48 hours
